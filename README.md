# Nouf Profile Webpage

This project is a personal profile webpage for Nouf Baabbad, showcasing her work as an Apple Academy Developer. The webpage is designed with a black and white color scheme and includes various sections to highlight her app and contact information.

## Project Structure

```
nouf-profile-webpage
├── src
│   ├── index.html          # Main HTML file
│   ├── styles              # Contains CSS styles
│   │   └── main.css        # Styles for the webpage
│   ├── scripts             # Contains JavaScript files
│   │   └── main.js         # JavaScript for interactions
│   ├── components          # Reusable HTML components
│   │   ├── Header.html     # Header section
│   │   ├── OurApp.html     # Our App section
│   │   ├── CodeBox.html    # Code box section
│   │   ├── Contact.html     # Contact section
│   │   └── Footer.html      # Footer section
│   └── assets              # Assets for the webpage
│       ├── icons           # Icons used in the webpage
│       │   ├── lume.svg    # App icon for Lume
│       │   └── academy-logo.svg # Academy logo
│       └── screenshots      # Screenshots of the app
│           ├── shot-1.svg  # First screenshot
│           ├── shot-2.svg  # Second screenshot
│           └── shot-3.svg  # Third screenshot
├── package.json            # npm configuration file
├── .gitignore              # Git ignore file
└── README.md               # Project documentation
```

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd nouf-profile-webpage
   ```

3. Install dependencies (if any):
   ```
   npm install
   ```

4. Open `src/index.html` in a web browser to view the webpage.

## Features

- **Header**: Displays the name and title with a button linking to the contact section.
- **Our App Section**: Showcases the app "Lume" with an icon, description, and screenshots.
- **Code Box**: Provides a link to the GitHub repository for the code.
- **Contact Section**: Contains links to LinkedIn, GitHub, and an email address.
- **Footer**: Displays the logo of the academy.
